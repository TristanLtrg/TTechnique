# TestTechnique

## Backend

accéder au fichier:

`cd todo-backend`

### Installation

installer:

`npm install`

fichier env:

créer un fichier .env en se basant sur le fichier `.env.example`

### Lancer le projet

Lancement:

`npm run start`

mode dev:

`npm run start:dev`

documnetation Api (Swagger):

`http://localhost:3001/api/`
